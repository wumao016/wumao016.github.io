# 构建工具和脚手架

# Gulp

Gulp 是一个自动化构建工具

安装

npm install --global gulp 全局安装

全局和进入项目中本地安装是都需要的

npm install --save-dev gulp 本地安装

步骤

  • npm init -y
  • 新建gulpfile.js配置
  • 如果是ES5+，还需安装 npm install --save-dev @babel/register @babel/core @babel/preset-env，重命名为 gulpfile.babel.js
  • 创建.babelrc 配置 {"presets": [ "@babel/preset-env" ]}
  • 运行 gulp

压缩合并js例子

  • npm init
  • 新建gulpfile.js
  • npm install gulp-uglify -D ,npm install gulp-concat -D
  • gulp
//gulpfile.js
var gulp=require('gulp');
var uglify=require('gulp-uglify');
var concat=require('gulp-concat');
var paths={
	scripts:['js/index.js','js/main.js']//找到js目录下的js
}
gulp.task('default',function(){
	gulp.src('js/*.js')
	    .pipe(uglify())
		.pipe(concat('all.min.js'))
		.pipe(gulp.dest('build'))//压缩，合并成all.min.js到build目录下
})

# Grunt

Grunt是javascript的构建工具

安装CLI

需要先将Grunt命令行（CLI）安装到全局环境中。安装时可能需要使用sudo权限来执行以下命令。

npm install -g grunt-cli
1

然后在文件夹中初始化 npm init -y；再在本地安装grunt

sudo npm install grunt --save-dev
1

之后，需要新建一个Gruntfile.js 相关配置就写在里面。下面根据官网来配置一个基本的。

module.exports=function(grunt){
	grunt.initConfig({
		pkg: grunt.file.readJSON('package.json'),
		//把相应的文件压缩
		uglify:{
			//找到静态目录
			static_mappings:{
				files:[{
					src:"js/index.js",
					dest:'build/index.min.js'
				},
				{
					src:"js/main.js",
					dest:'build/main.min.js'
				}]
			}
		},
		//合并文件
		concat:{
			bar:{
				src:["build/*.js"],
				dest:"dest/all.min.js"
			}
		}
	});
	//加载包含“uglify”任务的插件
	grunt.loadNpmTasks("grunt-contrib-uglify");
	grunt.loadNpmTasks("grunt-contrib-concat");
	//默认被执行的任务列表
	grunt.registerTask('default', ['uglify','concat']);
}

# Bower

Bower是包管理工具

安装

npm install -g bower

常用命令

  • bower install jquery 安装jquery
  • bower list 查看安装的库，如果有相关依赖的库版本不匹配，也会有提示
  • bower update jquery 更新
  • bower cache jquery 查缓存
  • bower info jquery 查jquery的版本
  • bower lookup jquery 打开官网

Yeoman 是脚手架工具，帮助我们创建项目。

安装

sudo npm install -g yo
