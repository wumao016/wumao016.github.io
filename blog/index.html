# 写博客的原因

主要为了记录自己学习的过程，能知道自己学过什么，常常复习，以免学完就忘。还有就是通过视频、书籍、实例来学习，如果再看一遍，会花费很多时间，把主要的记录下来，也省了很多时间。

新项目上传github步骤

1.进入Github首页，点击New repository新建一个项目
2.填写相应信息后点击create repository即可 
3.cd 本地项目
git init
git add README.md
git commit -m "first commit"
git branch -M master
git remote add origin git@github.com:wumao016/myVue.git
git push -u origin master
如果没有README.md可能会报错，要加一个再从新执行。
第二次上传
1.git add .
2.输入命令：git commit -m "XXXX" (XXX为注释 必须填)
3.输入命令：git push origin master
