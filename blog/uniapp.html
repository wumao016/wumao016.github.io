# Uni-app使用

官方文档地址 (opens new window)，官方IDE HBuilderX

# 入门

  • 通过官方IDE，新建一个uniapp项目，基础目录 pages里面是页面，static里面是静态文件，main.js入口文件，App.vue公共等。
  • 只要安装对应的开发工具，就能运行在相应的平台上。

组件

//父->子组件传值
//子组件
<template>
	<view>{{content}}</view>
</template>
<script>
	export default {
		props:{
			content:{
				type:String,
				default:"1"
			}
		}
	}
</script>
//父组件
<template>
	<textr content="111"></textr>
</template>
<script>
	import textr from "./content.vue"
	export default {
		...
		components:{
			textr
		}
	}
</script>
//全局注册组件
main.js
import textr from "./content.vue"
Vue.component("textr",textr)

页面导航

  • H5通过window.history 属性对其进行访问，改变路由记录从而实现跳转
  • ios/安卓 是改变根视图或操作导航控制器出栈进栈从而实现跳转
  • 小程序实现也是改变根视图或操作导航控制器出栈进栈从而实现跳转

uni-app 有两种页面路由跳转方式：使用navigator组件跳转、调用API跳转。
