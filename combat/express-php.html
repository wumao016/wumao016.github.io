# 用Express和PHP入门实战

用express和php，前台写用户名传到后台数据库的例子，算是最基础的前后台数据交互的例子，主要练习express框架，连接sql数据库。

1.首先，目录如下

node_modules 各个模块；public 静态文件，包括css和js；views 页面；app.js启动文件；package.json描述文件。

下面，记录一下步骤，以及代码：

  • 1.新建一个目录，node初始化，安装express，在appjs里写基本代码连通接口。
  • 2.再写容错相关代码以及支持加载静态文件相应代码。
  • 3.安装swig，以及在views里新建两个html模版，再appjs里启动使用swig，以及相关路由。
  • 4.在html里写一个输入框以及一个提交按钮。
  • 5.在js里写点击按钮发送ajax请求。
  • 6.安装mysql，并在appjs里写连接数据库代码。
  • 7.在数据库里新建一个表，接收数据。

具体代码：app.js

var express=require("express");
var app=express();
var mysql=require("mysql");
app.use(express.static('public'));
//swig
var swig=require("swig");
app.set("view engine","html");
app.engine("html",swig.renderFile);
//设置mysql
var connection=mysql.createConnection({
	host:"localhost",
	user:"root",
	password:"",
	database:"PHPlesson"
});
connection.connect();
//路由

app.get("/",function(req,res,next){
	res.render("index")
})
app.get("/receive",function(req,res,next){
	console.log(req.query.username);
	var post={
		username:req.query.username
	};
	var query=connection.query("INSERT INTO username SET ?",post,function(err,result){
		if(err){
			res.json({
				sucess:"no",
				mss:"插入失败"
			})
		}else{
			res.json({
				sucess:"ok",
				mss:"插入成功"
			})
		}
	})
})
//容错
app.get('*',function(req,res,next){
	res.status(404);
	res.end('404')
});
app.use(function(err,req,res,next){
	res.status(500);
	res.end('error')
})
app.listen(8082,function(){
	console.log("sever is running")
})
